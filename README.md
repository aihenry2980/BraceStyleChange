# BraceStyleChange
Change the brace style from the end of the line to next line
Written in Python 3.5

##Original file:
```cpp
if (true) {
 //do something
} else {
 //something else
}
```
##Changed file:
```cpp
if (true) 
{
 //do something
} 
else 
{
 //something else
}

```
